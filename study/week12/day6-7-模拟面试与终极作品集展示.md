# Week 12 Day 6-7: 模拟面试与终极作品集展示

**预计学习时间**: 10-12小时  
**难度**: ⭐⭐⭐⭐⭐

## 项目概述

这是整个 12 周 Web3 学习之旅的终点站。在这个周末，你将不再编写新的代码，而是作为一名"准 Web3 工程师"，整理你过去 3 个月的成果，打磨你的个人品牌，并进行一次全真模拟面试。这一步至关重要，它将把你的技术能力转化为可视化的职业竞争力。

## 第一部分: 终极作品集打磨 (4小时)

### 1. 项目大扫除 (Cleanup)

回顾你过去 12 周做过的所有项目（DeFi, NFT, DAO, Cross-chain, CI/CD...）。
挑选出 **3 个** 最具代表性的项目（建议：一个 DEX/Lending，一个 NFT Market，一个 DAO 工具）。

对这 3 个项目进行"产品级"的打磨：
-   **代码质量**: 删除 `console.log`，添加必要的 NatSpec 注释，运行 `prettier`。
-   **文档**: 确保 README.md 有清晰的架构图、部署地址和操作 GIF。
-   **演示**: 确保 Vercel/IPFS 的演示链接是可访问的。如果测试网 ETH 没了，去领水并重新部署。

### 2. 个人主页集成

如果你在 Day 2 创建了 Link3 或个人网站，现在将这 3 个项目更新上去。
确保你的 GitHub Pinned Repositories 也是这 3 个。

### 3. 录制 Demo 视频

为你的"旗舰项目"录制一个 3-5 分钟的高质量视频。
-   **开头**: "你好，我是 [名字]，这是我开发的 [项目名]，一个去中心化的..."
-   **中间**: 演示核心功能，顺便讲解一下遇到的技术难点（比如防重入、Gas 优化）。
-   **结尾**: "项目的代码已开源在 GitHub，欢迎查阅。"
-   **发布**: 上传到 YouTube/Bilibili，并嵌入到 GitHub README。

## 第二部分: 全真模拟面试 (4小时)

找一个朋友（最好是懂技术的），或者对着摄像头自问自答。请严格遵守时间限制。

### 环节 1: 自我介绍 (3分钟)
> "Tell me about yourself."

**参考模版**:
"我是一名拥有 [X] 年经验的软件工程师，最近一年专注于 Web3 开发。在过去的 3 个月里，我系统性地构建了从 ERC20 标准到复杂的 DeFi 借贷协议和跨链桥。
我最自豪的项目是一个基于 UUPS 代理模式的可升级 DAO 治理平台，我独立完成了合约编写、前端集成以及自动化测试流水线的搭建。
我对 EVM 底层原理和智能合约安全有浓厚的兴趣，这也是我申请贵公司 Smart Contract Engineer 职位的原因。"

### 环节 2: 技术深挖 (20分钟)
从你的简历中挑一个项目，进行深挖。

-   "为什么你在这个项目中选择了 Diamond 模式而不是 Transparent Proxy？"
-   "你是如何处理链上随机数生成的？"
-   "如果 Chainlink 预言机报价延迟了，你的借贷协议会发生什么？"
-   "你在这个项目中遇到的最大 Bug 是什么？怎么解决的？"

### 环节 3: 现场编码 (20分钟)
**题目**: 实现一个 `StakingRewards` 合约。
-   用户可以质押 Token。
-   每秒获得固定奖励。
-   编写 `stake`, `withdraw`, `getReward` 函数。
-   **加分项**: 考虑算法精度丢失问题，考虑 `updateReward` 修饰符模式。

### 环节 4: 行为与反问 (10分钟)
-   "你更喜欢远程工作还是办公室？"
-   "你对我们公司最近推出的 [产品X] 有什么看法？"
-   **你的反问**: "开发团队的代码审查流程是怎样的？"

## 第三部分: 求职冲刺 (2小时)

### 1. 简历最终检查 (Checklist)
- [ ] 是一页纸 PDF 吗？(不要 word)
- [ ] 名字下面就是 GitHub, LinkedIn, Email 链接吗？
- [ ] 技能点 (Skills) 是否列出了 Solidity, Hardhat, Ethers.js, React？
- [ ] 项目经历 (Projects) 是否有 Live Demo 链接？

### 2. 投递策略
不要海投。
1.  **Tier 1 (梦想公司)**: 3-5 家。深入研究他们的文档、GitHub、Discord。写一封极具针对性的 Cover Letter（甚至帮他们修一个 Bug）。
2.  **Tier 2 (意向公司)**: 10-20 家。根据职位描述微调简历关键词。
3.  **Tier 3 (练手公司)**: 若干。先面这些公司积累面试经验。

### 3. 渠道
-   **Web3.career / Cryptocurrency Jobs**: 传统投递。
-   **Twitter**: 搜索 "hiring solidity"，直接 DM 创始人或 CTO（带上你的 GitHub 和 Demo）。
-   **Discord**: 很多项目在社区招募 Contributor，先贡献再入职是成功率最高的路径。
