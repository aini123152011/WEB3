# Week 12 Day 5: 软技能与远程工作

**预计学习时间**: 4-6小时  
**难度**: ⭐⭐

## 学习目标

通过本节课程,你将能够:
- 掌握 Web3 行业高效的异步沟通技巧
- 理解并熟练使用 Web3 社区黑话 (Slang)
- 了解如何在 DAO 中通过贡献建立声誉
- 学习远程工作的自我管理与健康维持

## 课程内容

### 1. 高效的异步沟通 (Async Communication)

Web3 团队通常分布在全球各地，时差导致同步会议很难协调。**异步优先**是行业标准。

#### 1.1 沟通原则

-   **Over-communication (过度沟通)**: 在缺乏肢体语言和即时反馈的环境下，多写一点背景信息总是好的。
-   **No "Hello"**: 不要只发一句 "在吗" 或 "Hello"。直接说事："Hello, I'm blocked by issue #123, can you help review my PR?"。
-   **Public by Default**: 尽量在公开频道 (Discord/Slack Channels) 讨论技术问题，而不是私聊 (DM)。这样其他人也能看到并学习，也构成了知识库。

#### 1.2 常用工具

-   **Loom**: 录制 2-3 分钟的屏幕演示视频，解释复杂的 Bug 或功能。比打字快 10 倍。
-   **Notion / Linear**: 编写结构化的设计文档和任务票。
-   **GitHub Discussions**: 进行长周期的技术决策讨论。

### 2. Web3 英语与黑话 (Slang)

融入社区的第一步是听懂他们在说什么。

#### 2.1 社区用语
-   **GM / GN**: Good Morning / Good Night. 每天打卡，建立社区归属感。
-   **WAGMI**: We're All Gonna Make It. 牛市口号，互相鼓励。
-   **NGMI**: Not Gonna Make It. 形容错误的操作或看跌的心态。
-   **Degen**: Degenerate. 赌徒/激进投资者，Web3 原住民的自嘲。
-   **Alpha**: 早期的高价值信息（内幕消息）。
-   **FUD**: Fear, Uncertainty, Doubt. 散布恐慌言论。
-   **CT**: Crypto Twitter. 推特上的加密货币圈子。

#### 2.2 技术术语
-   **Bridging**: 跨链转账。
-   **Ape in**: 不做调研直接冲进去买。
-   **Rug Pull**: 项目方跑路。
-   **Diamond Hands**: 坚定持有，不卖。
-   **Paper Hands**: 一跌就卖。

### 3. DAO 贡献与声誉建设

DAO (Decentralized Autonomous Organization) 是 Web3 独特的组织形式。

#### 3.1 如何开始贡献 (Onboarding)

1.  **Join Discord**: 进入感兴趣的 DAO。
2.  **Read Docs**: 阅读 Manifesto, Governance Process。
3.  **Introduce Yourself**: 在 `#intro` 频道介绍你的技能。
4.  **Find Bounties**: 在 Dework, Gitcoin 或 DAO 内部的 Bounty Board 寻找小任务。

#### 3.2 治理参与

-   **Snapshot**: 参与链下投票。
-   **Forum**: 在 Discourse 论坛上发表提案或评论。
-   **Delegate**: 如果你持有 Token 但没时间投票，可以委托给专业的 Delegate。

#### 3.3 获得报酬

-   **Grants**: 申请拨款开发特定功能。
-   **Coordinape**: 贡献者之间互相打分，分配薪酬池 (Give circles)。
-   **Stream**: 使用 Sablier / Superfluid 接收流支付工资。

### 4. 远程工作健康指南

#### 4.1 时间管理

-   **Deep Work**: 设定每天 4 小时的不被打扰时间。
-   **Time Zones**: 使用 World Time Buddy 管理时差。明确你的 "Online Hours"。

#### 4.2 避免倦怠 (Burnout)

Web3 是 24/7 永不眠的市场，很容易让人焦虑。
-   **关闭通知**: 睡觉时把 Discord/Telegram 设为静音。
-   **Touch Grass**: 这是一个 Meme，意思是"出门走走，接触大自然"，不要整天盯着 K 线和代码。
-   **分离工作与生活**: 尽量不要在卧室工作。

## 实践练习

### 练习1: 模拟异步沟通

假设你在做一个开源项目，遇到一个依赖库的 Bug。请编写一段请求帮助的信息（英文），包含：
1.  问题描述 (What happened)
2.  复现步骤 (How to reproduce)
3.  你尝试过的解决方案 (What you tried)
4.  期望的帮助 (What you need)

### 练习2: DAO 探索

1.  加入 [Developer DAO](https://www.developerdao.com/) 或 [Buildspace](https://buildspace.so/) 的 Discord。
2.  找到他们的治理论坛。
3.  阅读最近的一个提案，并理解其争议点。

### 练习3: 录制 Loom

选择你之前做过的一个项目，录制一段 3 分钟的英文介绍视频。
-   "Hi, I'm [Name]. This is my project [Project Name]."
-   "It solves [Problem]."
-   "Let me show you how it works..." (演示操作)
-   "The tech stack includes..."
