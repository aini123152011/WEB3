# Web3测试工程师学习计划 - 12周速成

## 📋 学习目标

通过12周系统学习，达到Web3测试工程师求职水平，具备以下能力：
- ✅ 能阅读和理解Solidity智能合约代码
- ✅ 能编写基础智能合约（ERC20、NFT等）
- ✅ 能使用Hardhat/Foundry进行合约测试
- ✅ 能识别常见智能合约安全漏洞
- ✅ 能测试DApp前端（钱包集成、交易流程）
- ✅ 能使用Web3.py/Ethers.js编写自动化测试
- ✅ 有3-5个完整的项目案例用于面试展示
