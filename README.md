## ⚠️ 重要提示

### 安全警告

- ⚠️ **永远不要分享你的私钥或助记词**
- ⚠️ **测试环境和主网环境严格分离**
- ⚠️ **合约上主网前必须经过充分测试和审计**
- ⚠️ **小心钓鱼网站和诈骗项目**
- ⚠️ **不要在公开代码中硬编码私钥**
